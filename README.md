# Dotfiles

My dotfiles for my Arch Linux setup.

## My setup

- **OS**: Arch Linux
- **WM**: Hyprland
- **Terminal**: Alacritty
- **Shell**: Bash
- **File Manager**: Thunar
- **Browser**: Brave
- **Font**: MesloLGS NF (Nerd Font)
