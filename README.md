# Dotfiles

My dotfiles for my Arch Linux setup.

## My setup

- **OS**: Arch Linux
- **WM**: Hyprland
- **Terminal**: Alacritty
- **Shell**: Bash
- **Editor**: Nano
- **File Manager**: Thunar
- **Browser**: Firefox
- **Music Player**: Spotify (w/ spicetify)
- **Font**: MesloLGS NF (Nerd Font)
- **GTK/QT5-6 Theme**: Dracula
- **Icons**: Tela
